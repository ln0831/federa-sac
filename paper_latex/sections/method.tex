\section{Method}

The paper studies the existing FedGrid runtime bundle rather than introducing a new controller from scratch. The active execution chain includes the root-level runner, training, deterministic evaluation, aggregation, table export, figure generation, and report generation scripts. The method family currently includes a non-federated baseline, topology-prototype sharing, clustered distillation, clustered no-distillation, gentle clustered variants, and robustness-oriented stress variants.

The current methodological contribution is expected to emphasize deterministic paired evaluation and context alignment. This section will be expanded after the code-review report is merged and the exact method description is cross-checked against the implementation.
